# Recruitment Intelligence — CloudWalk Assignment

Data analysis of a fictional recruitment dataset (601 candidates, 8 channels, 6 recruiters).

## What's included
- `recruitment_dashboard.html` — Interactive dashboard with AI assistant (open in any browser)

## How to use the dashboard
Download the HTML file and open it in your browser. No setup needed.
You can update the dashboard by adding a new Excel file.

## Key findings
- GitHub and Inbound are the top-converting channels (11% and 10.1%)
- 34.9% no-response rate is the single biggest funnel leak
- A composite assessment score ≥80 reliably predicts hire outcomes
- Top recruiter converts at 4× the rate of the lowest — a coaching opportunity
