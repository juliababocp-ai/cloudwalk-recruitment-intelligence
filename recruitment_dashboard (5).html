Key Insights
01.
Funnel Efficiency
Only 8.2% of sourced candidates convert to hire. The sharpest drop is at first contact — 34.9% never respond. Once engaged, 75.5% of responders pass screening.
02.
Best Channels
GitHub (11%) and Inbound (10.1%) lead all channels — outperforming average by 30%+. GitHub candidates arrive pre-vetted via public work. Inbound signals genuine intent.

Scores Predict Outcome

Seniority Drives Conversion

Conversion Signal Framework
High Probability Signals
  • Technical score ≥ 80
  • Behavior score ≥ 78
  • Manager score ≥ 79
  • Response within 3 days
  • Staff or Senior seniority
  • GitHub or Inbound source
  • Híbrido work mode preference
Persist with Caution
  • Technical score 70–80
  • No response after 7 days — try one more touch
  • Timing mismatch — follow up in 30–45 days
  • Pleno with strong domain experience
  • LinkedIn passive (screen well if engaged)
Low Conversion Signals
  • No response after 10+ days and 3 touches
  • Technical score below 70
  • Low engagement flagged by recruiter
  • Salary mismatch surfaced early
  • Indicação with no warm intro context
  • Junior + Remote combination
When to Persist
  • High scorer who missed first outreach
  • Timing mismatch — nurture and revisit
  • Headcount closed — hold in talent pool
  • LinkedIn passive with matching senior profile
  • Strong referrer endorsement with context
